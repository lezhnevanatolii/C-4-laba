# C-4-laba
Отчёт по Лабораторной работе №4 на тему «Перегрузка операторов»
Лабораторная работа №4.

Задание 1. 
Реализовать определение класса (поля, свойства, конструкторы (не менее трех), перегрузку оператора вывода для вывода полей, заданный метод согласно варианту). Протестировать все методы, включая конструкторы, все исходные данные вводятся с клавиатуры, не забывайте проверять данные на корректность.
Название класса:		
QuadraticEquation	
Поля:
double a
double b
double c
(коэффициенты)	
Методы: 
Вычисление корней квадратного уравнения. Результат должен быть массивом величин типа double (в нем от 0 до 2-х элементов, в зависимости от количества корней).

 Алгоритм решения задачи
1.	Объявить и заполнить исходный набор данных — вектор целых чисел, например, с помощью инициализации или пользовательского ввода.
2.	Создать дек D, использовав итераторы из исходного массива или вектора.
3.	Определить половину длины дека — size / 2, где size — общий размер дека.
4.	Вывести первую половину элементов дека в обратном порядке:
— от rbegin() до rbegin() + half.
5.	Вывести вторую половину элементов дека в обратном порядке:
— от rbegin() + half до rend().
6.	Форматировать вывод.

Структура кода
•	Инициализация исходных данных
•	Создание дека из исходных данных
•	Получение половины размера
•	Вывод первой половины в обратном порядке
•	Вывод второй половины в обратном порядке

Ожидаемый вывод:
Квадратное уравнение: 1x^2 + 0x + 1
Уравнение не имеет действительных корней.

Задание 2. 
Добавить к реализованному в первом задании классу указанные в варианте перегруженные операции. В главной функции main показать работу всех функций с дружественным интерфейсом.
Название класса	
QuadraticEquation	
Методы:
Унарные операции: ++ увеличивает коэффициенты уравнения на 1; -- уменьшает коэффициенты уравнения на 1. 
Операции приведения типа: double (неявная) – результатом является дискриминант уравнения. bool (явная) – результатом является true, если корни существуют и false в противном случае;
Бинарные операции == QuadraticEquation t — уравнения равны, если равны их коэффициенты; != QuadraticEquation t — уравнения не равны, если не равны их коэффициенты

Алгоритм решения
1.	Ввод коэффициентов:
•	Запросить у пользователя значения a, b, c.
2.	Создание объекта уравнения:
•	Использовать введённые коэффициенты для создания объекта класса QuadraticEquation.
3.	Вывод уравнения:
•	Отобразить введённое уравнение.
4.	Демонстрация операторов:
•	Применить префикс и постфикс инкремента (++) и декремента (--), отображая результаты.
5.	Преобразование в double:
•	Вычислить дискриминант (b^2 - 4*a*c) и вывести его.
6.	Проверка наличия действительных корней:
•	Использовать преобразование в bool.
7.	Вычисление корней:
•	Вызвать метод getRoots(), который выводит корни уравнения.
8.	Проверка операторов равенства:
•	Создать второе уравнение, сравнить его с исходным, вывести результат.

Структура кода
•	Ввод коэффициентов a, b, c
•	Создание объекта QuadraticEquation
•	Вывод уравнения
•	Демонстрация оператора ++ (префикс), вывод
•	Демонстрация оператора ++ (постфикс), вывод
•	Демонстрация оператора -- (префикс), вывод
•	Демонстрация оператора -- (постфикс), вывод
•	Преобразование в double и вывод дискриминанта
•	Проверка функции bool (есть ли корни), вывод
•	Вызов метода getRoots() для вывода корней
•	Создание второго уравнения для сравнения
•	Сравнение и вывод результата
